建立 Date() 物件

在 JavaScript 中取得時間資訊需使用 Date() 物件來指向某一個時間點，透過物件內建的函式來取得相關資訊。
取得時間的資訊有兩種方式，第一種是透過 Date.now() 來取得時間戳記（tiemstamp,毫秒），第二種是透過建構式建立 Date()物件，物件會回傳目前時間資訊的日期物件，然後就可以利用內建的函式來取得資訊。
